./index.json

Datos completos de la semana más reciente incluyendo reuniones, materiales y estadísticas.

🗂️ Datos Específicos

./data/{año}/week-{número}.json

Datos específicos de una semana en particular.

🚀 Uso de la API

Todos los endpoints devuelven datos en formato JSON. Puedes acceder directamente a las URLs o usar herramientas como fetch, axios, etc.

📝 Ejemplo de uso:

// Obtener la semana más reciente
fetch('https://neri-gut.github.io/jw-scraping/latest.json')
  .then(response => response.json())
  .then(data => console.log(data));

// Obtener lista de semanas
fetch('https://neri-gut.github.io/jw-scraping/weeks.json')
  .then(response => response.json())
  .then(data => console.log(data.weeks));
